# webken-11-handson

初期設定・起動の方法

## HTML&CSS
**起動方法**  
- 1,htmlファイルを右クリック
- 2,エクスプローラーで表示する

## DOM 
**起動方法**  
- HTML&CSSと同じ

## Node
**初期設定**
- 1,`Node/server`にcd
- 2,`npm install`を実行

**起動方法**
- `npm start`

## Hono
**初期設定**
- 1,`Hono/server`にcd
- 2,`npm install hono @hono/node-server`を実行

**起動方法**
- `npm start`

## SQL
**初期設定**
- 1,`SQL/server`にcd
- 2,`npm install better-sqlite3`を実行

**起動方法**
- `npm start`
